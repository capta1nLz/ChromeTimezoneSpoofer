# Privacy Policy

**Last updated:** April 24, 2026

This Privacy Policy applies to the Chrome extension **Chrome Timezone Spoofer**.

## No Data Collection

**Chrome Timezone Spoofer does not collect, transmit, sell, share, or store any personal or sensitive user data.**

The extension does not collect any of the following:

- Personally identifiable information
- Account information
- Passwords
- Cookies
- Authentication tokens
- Financial information
- Health information
- Browsing history
- Website content
- Form data
- User-generated content
- URLs visited by the user

## Local Extension Settings

The extension uses `chrome.storage.local` only to save local extension preferences on the user's own browser, including:

- `enabled`: whether the extension is enabled or disabled
- `timezone`: the timezone selected by the user
- `language`: the interface language selected by the user

These settings are stored locally in the user's browser. They are not uploaded, transmitted, sold, shared, or sent to the developer or any third party.

## Content Script Behavior

The extension may run a content script on websites for the sole purpose of applying timezone spoofing logic related to JavaScript `Date` and `Intl.DateTimeFormat`.

The content script does not read, collect, copy, store, or transmit webpage DOM content, form inputs, account information, cookies, browsing history, URLs, or user identity information.

## Timezone List Request

The extension popup may request a public timezone list from GitHub Raw in order to display available timezone options.

This request is only used to retrieve public timezone data. The extension does not send user settings, website content, identity information, browsing data, or any personal data to GitHub, the developer, or any third-party server.

## No Tracking or Analytics

The extension does not use analytics, advertising trackers, telemetry, remote logging, or behavioral tracking.

## No Data Sharing

The extension does not sell, rent, transfer, or share user data with any third party.
